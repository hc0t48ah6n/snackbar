M3Snack.js

Material 3 デザインのスナックバーをJavaScript一行で。CSS不要、Promise対応。

ライブデモ

ボタンをクリックしてスナックバーの動作を確認してください。

インストール

HTMLの <head> 内に以下のタグを貼り付けるだけで準備完了です。

<script src="https://hc0t48ah6n.github.io/snackbar/snackbar.js"></script>

基本的な使い方

M3Snack.show() 関数にオプションオブジェクトを渡します。

// 1. シンプルな通知
M3Snack.show({
  message: "保存完了"
});

// 2. アクションボタンと処理
const clicked = await M3Snack.show({
  message: "削除しました",
  actionText: "元に戻す",
  duration: 5000
});

if (clicked) {
  // ボタンが押された時の処理
  M3Snack.show({ message: "復元中..." });
}
